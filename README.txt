PAYROLL PT STAR SAMUDERA LOGISTIK - REVISI DOSEN

FITUR UTAMA
- Kode jabatan dan NIP otomatis.
- Rekap absensi bulanan, bukan presensi harian.
- Edit absensi membutuhkan persetujuan pimpinan.
- Tarif lembur Rp15.000 per jam.
- Potongan alpha Rp25.000 per hari.
- Tidak ada tunjangan, pajak, BPJS, menu potongan, atau menu slip gaji.
- Preview hitungan gaji sebelum payroll disimpan.
- Status Sudah Dibayar/Belum Dibayar.
- Cetak rincian gaji melalui menu Proses Payroll.
- Laporan bergrafik dengan filter 1 bulan, 2 bulan terakhir, dan 1 tahun terakhir.
- Forgot password OTP mode uji coba.

CARA FRESH INSTALL
1. Salin folder ke C:\xampp\htdocs\payroll_star_final
2. Start Apache dan MySQL.
3. Import database.sql melalui phpMyAdmin.
4. Buka http://localhost/payroll_star_final

CARA UPGRADE DATABASE LAMA
1. Backup folder project dan ekspor database lama.
2. Ganti file project dengan versi revisi ini.
3. Buka http://localhost/payroll_star_final/install_or_upgrade.php
4. Setelah selesai, hapus install_or_upgrade.php.
